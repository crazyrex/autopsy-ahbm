The feature extraction algorithm in BuflDiff is based on the algorithm 
in sdhash, a tool release under Apache Liscense, Version 2 written by 
Vassil Roussev and Candice Quates. See http://sdhash.org

The tool uses sdhash for measuring similarity, however neither it source 
nor binary distirubtion is provided with this project.
